Stabilized Technical Report

UTR #36, “Unicode Security Considerations”, has been stabilized: There are no plans to publish another update for it. The last version can be found at https://www.unicode.org/reports/tr36/tr36-15.html

This Technical Report has not been maintained since 2014. Some material has been superseded by UTS #39 “Unicode Security Mechanisms” and UTS #55 “Unicode Source Code Handling”, and by updates to UAX #31 “Unicode Identifiers and Syntax”; recommendations made in UTR #36 may conflict with the ones given in these more up-to-date standards or may otherwise be misleading. Some material may still be useful, and may be extracted in the future for use in other specifications.
